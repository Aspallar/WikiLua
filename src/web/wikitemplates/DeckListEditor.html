Select the deck list that you wish to add a deck to

 

The following dropdown lists decks that have no entry in the deck lists. To get started select the deck that you want to add from the dropdown.

[[File:Redirected.png|link=]]

The page for the selected deck is a [[Help:Redirect|redirect]] to another page.

Redirected deck pages should not be added to a deck list.

Enter the name for your deck as it is to appear in the deck list. This can be different to the page name for the deck, but should be similar to it. We recommend that you don't change this unless you have good reason to.

Enter the type or strategy of deck. Select it from the dropdown or type it if your deck type does not appear in the dropdown.

Enter the author of the deck. If the author has an account on the wiki use their user name.

Enter a brief description of your deck. You should keep it fairly short as long descriptions do not look good on a deck list. For example Easy to pull off combo with control elements.

Tick the colors used in the deck. This will have been initialized from the deck definition, however if your deck contains cards that require colorless mana to cast you will have to tick "colorless" manually. Don't tick colorless just for artifacts.

Tick the following box if the deck comes with a sideboard.

When you are sure the above details are correct, click this button to add the deck to the deck list.
